# grb256

Neovim-compatible colorscheme originally developed by
[Gary Benhardt](https://github.com/garybernhardt) and my preferred theme for a
very long time.

<img width="2520" height="1342" alt="image" src="https://github.com/user-attachments/assets/c50fb356-8d1c-4ff9-9f42-a960b8b5ae56" />

```lua
-- lua/plugins/colorscheme.lua
return {
  { "vcavallo/grb256" },
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "grb256",
    },
  },
}

```
